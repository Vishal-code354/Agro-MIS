# Agro-MIS
Developed an intelligent Agriculture Management Mobile Application designed to assist farmers with real-time insights, commodity analytics, and smart decision-making tools using secure cloud-based authentication and data integration.

# Key Features:
1.	Secure User Authentication: Implemented login/registration with token-based authentication using ASP.NET Core API and Firebase for secure user identity management.
2.	Real-Time Weather & AQI: Provides live weather forecasts and daily AQI data across all major Indian regions.
3.	AI Chatbot Assistance: Integrated chatbot offering guidance on crop selection, fertilizer usage, irrigation, and pest control.
4.	Government Commodity Data: Automated integration of ISM government datasets for daily commodity prices, fertilizer demand–supply–consumption, and market fluctuations.
5.	Crop Production Insights: Offers crop production trends and analytics to support better planning and yield optimization.
6.	Weekly Agriculture News: Curates weekly agriculture news, policies, and government updates.
7.	Modern Mobile UI: Built a responsive and intuitive Flutter interface optimized for all device sizes.
